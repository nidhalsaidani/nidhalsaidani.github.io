Context and activity recognition for smart glasses users using AI and a smart badge

Project duration: October – December 2023

Project context

This academic project was carried out at Sorbonne University in collaboration with the LiP6 laboratory and EssilorLuxottica. Our team aimed to explore intelligent context awareness for wearable technologies, with a focus on smart glasses applications.

Project overview

We developed a prototype system capable of recognizing both environmental context and user activity using a smart wearable badge. The badge integrates multiple sensors — a camera, microphone, and IMU — and processes their signals using deep learning techniques. The solution was deployed on a compact embedded device (Nicla Vision), enabling real-time and energy-efficient AI for wearable use cases.
