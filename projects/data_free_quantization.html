Data-Free Quantization

Project duration: January – March 2024

Project context

This project was developed as part of an academic research effort at Sorbonne University, led by Nidhal Saïdani in collaboration with team members Assil Mahfoudi, Abdelkader Amroun, and Hania Djamai. The main goal is to explore and implement Data-Free Quantization (DFQ) techniques, which enable neural networks to be quantized efficiently without access to original training data, facilitating deployment on resource-constrained devices.

Project overview

The DFQ method reduces model size and computational requirements by quantizing weights, biases, and activations to low bit-widths, while maintaining high accuracy. Our implementation includes multiple techniques such as bias correction, weight clipping, and dynamic quantization measurement. The project demonstrates effectiveness on classification and segmentation tasks using popular models such as MobileNetV2 and DeepLabV3+.

Key features

  • Data-free quantization for deep neural networks (no access to original training data required)
  • Quantization of weights, activations, and biases with 8-bit precision
  • Bias correction and weight clipping to improve accuracy
  • Support for classification and segmentation tasks
  • Visualization of weight distributions before and after quantization
  • Easy environment setup with virtualenv and conda
